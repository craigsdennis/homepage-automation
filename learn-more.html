This experiment is built using a smattering of Twilio offerings. First and foremost, this is leaning very heavily on AutoPilot. This is what creates the bot like experience through many channels of communication.

This project is also Serverless. I'm hosting on GitHub pages, and making use of Twilio Functions.

The realtime data is being provided via a product known as Twilio Sync. Any record added to the Sync list is then handled by any functions that subscribe to specific commands.

The code is available on GitHub.
